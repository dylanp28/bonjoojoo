## 1. LUXURY PHOTOGRAPHY DIRECTION

### HERO PHOTOGRAPHY STANDARDS

#### Lighting Setup: "Museum Quality"
```
Primary Setup:
• Key Light: 5500K LED panel with softbox (45° angle)
• Fill Light: Reflector panel for shadow control
• Accent Light: Focused spot to enhance diamond fire
• Background Light: Gradient wash for depth

Specialized Diamond Lighting:
• Fiber optic ring light for maximum scintillation
• Polarized filters to control reflections
• Color temperature precisely calibrated for diamond color
• Multiple exposure bracketing for HDR processing
```

#### Composition Rules: "Architectural Elegance"
1. **The Golden Ratio**: All product placement follows φ = 1.618
2. **Breathing Room**: Minimum 40% negative space around hero subjects
3. **Leading Lines**: Subtle directional elements guide eye to diamonds
4. **Depth Layering**: Minimum 3 visual planes (foreground, subject, background)

#### Background Concepts
```
Concept 1: "Pure Minimalism"
• Seamless white to warm white gradient (5° temperature shift)
• Subtle shadow casting for dimensionality
• No visible texture or pattern

Concept 2: "Architectural Luxury" 
• Carrara marble texture (subtle, never overpowering)
• Brass accent lines as compositional elements
• Natural light simulation through gradient

Concept 3: "Modern Atelier"
• Clean laboratory/workshop aesthetic
• Precision tools as artistic elements
• Technology meets craftsmanship narrative
```

### LIFESTYLE PHOTOGRAPHY: "Conscious Luxury Living"

#### Model Direction
```
Casting Guidelines:
• Age Range: 25-45 primary, 22-28 secondary
• Aesthetic: Natural beauty, minimal retouching
• Diversity: Global luxury appeal across ethnicities
• Expression: Confident serenity, not overt happiness

Styling Requirements:
• Wardrobe: Contemporary luxury (Celine, Acne, COS aesthetic)
• Hair/Makeup: Editorial quality but natural finish
• Setting: Architectural spaces, natural light
• Interaction: Jewelry worn, not displayed
```

#### Setting Concepts
```
Location 1: "California Modern"
• Mid-century modern architecture
• Clean lines, floor-to-ceiling windows
• Natural materials: wood, stone, leather
• Golden hour lighting

Location 2: "Urban Sophistication"
• Gallery spaces, museums, design studios
• Concrete, steel, and glass elements
• Controlled artificial lighting
• Monochromatic color schemes

Location 3: "Natural Luxury"
• High-end outdoor settings
• Botanical gardens, luxury hotels
• Soft, diffused natural lighting
• Organic textures as contrast to jewelry precision
```

### PRODUCT PHOTOGRAPHY SPECIFICATIONS

#### Technical Requirements
```
Resolution: Minimum 4K (3840 x 2160) for all hero images
Color Space: sRGB for web, Adobe RGB for print
File Format: RAW capture, optimized WebP for web delivery
Bit Depth: 16-bit processing, 8-bit final delivery

Specific Shots Required per Product:
1. Hero shot: 45° angle showing maximum brilliance
2. Detail shot: 10x magnification showing craftsmanship
3. Profile view: Side angle showing setting architecture  
4. Scale shot: Product with size reference object
5. Lifestyle shot: Product worn in elegant context
```

#### Post-Production Standards
```
Color Grading:
• Maintain diamond color accuracy (no artificial enhancement)
• Consistent color temperature across all images
• Preserve metal color fidelity
• Subtle contrast enhancement without oversaturation

Retouching Guidelines:
• Remove dust and surface imperfections only
• Maintain realistic surface textures
• No artificial sharpening or clarity enhancement
• Preserve natural diamond inclusions where visible
```

## 3. TYPOGRAPHY SYSTEM

### LUXURY TYPOGRAPHY HIERARCHY

#### Primary Font: Custom or Premium Alternative
```
Brand Headlines: "Optima" or custom geometric serif
• Elegant, architectural letterforms
• Excellent diamond/jewelry industry precedent
• Available weights: Light, Regular, Medium, Bold

Body Text: "Avenir Next" or "Proxima Nova"
• Swiss precision meets approachability
• Excellent readability across all devices
• Complete weight and style family

Accent Font: "Didot" or "Bodoni" (sparingly)
• High contrast for luxury emphasis
• Fashion/luxury industry standard
• Use only for special occasions/quotes
```

#### Type Scale: "Golden Ratio Based"
```css
/* Typography Scale */
--text-xs: 0.75rem;     /* 12px - Fine print, captions */
--text-sm: 0.875rem;    /* 14px - Support text, labels */
--text-base: 1rem;      /* 16px - Body text baseline */
--text-lg: 1.25rem;     /* 20px - Large body text */
--text-xl: 1.5rem;      /* 24px - Subheadings */
--text-2xl: 2rem;       /* 32px - Section headings */
--text-3xl: 3rem;       /* 48px - Page headings */
--text-4xl: 4rem;       /* 64px - Hero headings */
--text-5xl: 6rem;       /* 96px - Display text */

/* Luxury Spacing */
--spacing-tight: 1.25;  /* Tight line height for headlines */
--spacing-normal: 1.5;  /* Standard body text */
--spacing-loose: 1.75;  /* Relaxed, luxury reading experience */
```

#### Typography Usage Guidelines
```
Hero Headlines:
• Font: Optima Light or similar
• Size: text-4xl to text-5xl
• Line Height: spacing-tight
• Letter Spacing: 0.02em (slightly open)

Product Names:
• Font: Optima Regular
• Size: text-xl to text-2xl
• Case: Sentence case preferred

Body Copy:
• Font: Avenir Next Regular
• Size: text-base (16px minimum)
• Line Height: spacing-loose (luxury reading)
• Paragraph spacing: 1.5em

Luxury Quotes:
• Font: Didot Regular (sparingly)
• Size: text-lg to text-xl
• Style: Italic for emphasis
• Color: Charcoal Deep
```
