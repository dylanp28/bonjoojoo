## 7. Campaign Operations

### Outreach Timeline

| Day | Action |
|-----|--------|
| Day 1 | Send DMs to Priority Wave 1 (contacts 1–5) |
| Day 3 | Send DMs to Wave 2 (contacts 6–10) |
| Day 6 | Follow up with Wave 1 non-responses (once only) |
| Day 8 | Send DMs to Wave 3 (contacts 11–20) |
| Day 11 | Follow up with Wave 2 non-responses |
| Day 14 | Evaluate responses; escalate top responders to paid or affiliate |
| Day 21 | Post-gift check-in; confirm posting timeline for gifted influencers |
| Day 30 | Review: who posted, coupon code performance, affiliate signups |

### Gift Budget

- **Gift value:** $200/influencer × 20 = $4,000 max (not all will accept)
- **Expected acceptance rate:** 30–40% → ~$800–$1,600 actual gift spend
- **Coupon codes:** Format `[HANDLE]15` (15% off for audience, tracked for attribution)

### Affiliate Program Setup Checklist

- [ ] Set up affiliate tracking software (Refersion, Impact, or ShareASale)
- [ ] Create affiliate landing page at bonjoojoo.com/affiliates
- [ ] Configure 30-day cookie tracking
- [ ] Build affiliate dashboard with real-time earnings view
- [ ] Set up monthly payout automation
- [ ] Prepare affiliate welcome kit (lookbook, brand guidelines, product images, UTM link generator)
