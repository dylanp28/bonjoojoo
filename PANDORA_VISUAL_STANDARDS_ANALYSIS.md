# PANDORA VISUAL STANDARDS ANALYSIS
## Completed Visual Research for Bonjoojoo Asset Creation

### PHOTOGRAPHY STYLE ANALYSIS

**Lighting & Composition:**
- Professional studio lighting with controlled reflections
- Pure white backgrounds (#FFFFFF) for product isolation
- Soft shadow casting that enhances dimensionality
- Diamond brilliance maximized through precise lighting angles
- High-resolution photography (minimum 1200x1200 product shots)

**Color Temperature:**
- Cool, neutral lighting temperature (~5500K-6500K)
- Consistent color reproduction across all product images
- No warm color casts that could affect diamond perception
- Crisp, clean aesthetic with maximum contrast

**Product Angles:**
- Multiple views per product (front, side, angled)
- Detail shots focusing on diamond settings and craftsmanship
- Lifestyle shots showing jewelry being worn
- 360-degree view capabilities for key pieces

### LAYOUT & UI STANDARDS

**Product Grid:**
- 3-4 column responsive grid layout
- Consistent product card dimensions
- Hover effects revealing additional product views
- Clear pricing hierarchy with sale pricing prominence

**Typography Hierarchy:**
- Clean sans-serif fonts for navigation and pricing
- Elegant serif fonts for headlines and brand messaging
- Consistent spacing and sizing across all elements
- High contrast text for accessibility

**Badge System:**
- "BEST SELLER" - Red/pink highlighting top performers
- "NEW" - Blue/teal for new arrivals
- "FINAL SALE" - Yellow/orange for clearance
- "LIMITED EDITION" - Purple for exclusive items

**Navigation Elements:**
- Mega-menu dropdown structure
- Breadcrumb navigation
- Filter and sort functionality
- Wishlist heart icons throughout

### BRAND POSITIONING ELEMENTS

**Luxury Indicators:**
- Premium photography quality
- Sophisticated color palette
- Clean, uncluttered layouts
- Professional model photography
- High-quality video backgrounds

**Trust Building:**
- Clear pricing without hidden costs
- Prominent return policy messaging
- Warranty information visibility
- Professional certification displays
- Customer service accessibility

### TECHNICAL SPECIFICATIONS

**Image Requirements:**
- Minimum 1200x1200px for product images
- 1920x1080px for hero banners
- WebP format for web optimization
- 2x and 3x retina display support
- Lazy loading for performance

**Color Standards:**
- sRGB color space for web consistency
- High contrast ratios for accessibility
- Consistent color reproduction
- Professional color grading
